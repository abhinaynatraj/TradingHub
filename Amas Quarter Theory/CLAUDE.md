# Amas Quarter Theory

Live TradingView Pine v6 indicator that applies bootcamp price-action concepts (15m quarters, 1h doji/line, 3h triad line/apex, 05-box bands, midlines) on the chart, with empirical probability readouts pre-computed from `nq_1m` / `es_1m`.

## Stack
- Python 3.14 · DuckDB 1.4.4 · pandas
- Pine v6 (single-file indicator)
- Reuses Amas Models patterns (`stats.wilson_ci`, `db.py`, `OUTCOME_MAX_BARS=1440`)

## Running

```bash
python3 engine/build.py                 # full pipeline
python3 -m pytest tests/ -q             # tests
```

## Database

Read-only from `../Fractal Sweep/candle_science.duckdb`. Tables `nq_1m`, `es_1m`. Always convert to NY tz at SQL layer.
