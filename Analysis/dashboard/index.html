NQ Hourly Analysis

About this dashboard
Two studies on intraday NQ price behavior, built from 1-min futures bars since 2014.
Each hourly bar is examined two ways: after it breaks the prior hour's range (Study 1), and inside the hour split into 15-minute quarters (Study 2). The previews below show the headline finding from each — open a study for the full breakdown.

How the dataset is built
From shared 1-minute NQ bars to study-ready parquet, in three steps.
  1. 1-min bars live in the shared candle_science.duckdb at nq_1m (timestamps in America/Toronto, converted to America/New_York for hour grouping).
  2. Hourly aggregation in Analysis/engine/bars.py drops any hour without 60 complete minutes (~9.6k incomplete hours skipped).
  3. Study engines (breakout_study.py, quarter_study.py) emit one parquet per slice into Analysis/data/{breakout,quarters}/; the dashboard reads them via DuckDB-Wasm in the browser.

Re-run the pipeline anytime with python3 Analysis/engine/run_all.py. The manifest above shows the last refresh.
